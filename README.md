## Project introduction / 프로젝트 소개

코로나 때문에 자동차를 이용하는 사람들이 많아지게 되었습니다.    
평소 일상에서 사용해왔던 렌트카 어플리케이션들의 장단점을 생각해보고,   
장점만을 모은 렌트카 서비스를 제공하고 싶었습니다.   
   
또한 남녀노소 누구나 직관적으로 쉽게 접근할 수 있는 서비스가 필요하다고 느끼게 되어   
이번 세미 프로젝트로 진행하게 되었습니다.
